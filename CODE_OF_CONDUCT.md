# Code of Conduct

## Our standards

ClaudeLander adopts the **Contributor Covenant**, version 2.1, as its code
of conduct.

The full text of the Code of Conduct is hosted by the upstream Contributor
Covenant project and is available at:

  https://www.contributor-covenant.org/version/2/1/code_of_conduct.html

We adopt that document by reference. By participating in this project --
filing an issue, opening a pull request, commenting on a discussion, or
contributing in any other form -- you agree to abide by it. The same
expectations apply to maintainers.

## Reporting

Reports of unacceptable behaviour should be sent to the project maintainer:

  pomzm67@gmail.com

Reports are handled in confidence. The maintainer will acknowledge each
report within a reasonable response time and follow the enforcement
guidelines published alongside Contributor Covenant 2.1.

For security vulnerabilities, please follow `SECURITY.md` instead -- those
go through a separate, private channel and should not be filed as a public
issue.
